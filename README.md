Layout Website

- il progetto sarà pubblicato su github in una repository
- bootstrap lato frontend installato con npm
- google font
- fontawesome
- file style.css personale
- header comune per tutte le pagine
- footer comune per tutte le pagine
- ogni pagina scelta dagli elementi nella navbar (chi siamo, prodotti, contatti,..)
    porterà ad un file dedicato php dove avrò header e footer importati
    e una propria sezione della pagina
- nella navbar un bottone ci porterà ad una pagina di login nella quale
    sarà presente un form per poter eseguire l'accesso simulato
    tramite inserimento di user e password
    in seguito se l'accesso sarà effettuato correttamente si entrerà in una pagina protetta dashboard.php
    se invece l'utente sbaglia l'accesso verrà reindirizzato ad una pagina di non accesso
